# Bitcoin History Reference

A comprehensive, continuously updated reference document covering the key people, organizations, events, and cultural moments in Bitcoin's history — from its cypherpunk intellectual precursors through the present day.

I am collecting references to people and projects in creating a sort of living history of Bitcoin. This is an ongoing project that is AI-assisted, so I can't guarantee accuracy. I did limit the editorializing in my prompts, so hopefully the slop is kept to a minimum. Other researchers may find the document useful.

The document currently has 344 entries across three tiers:

- **Tier 1** — Foundational figures (cypherpunk precursors, Satoshi, the earliest contributors)
- **Tier 2** — Builders (developers, exchanges, wallets, Lightning, hardware, infrastructure)
- **Tier 3** — Catalysts (investors, media, politicians, circular economies, antagonists, cultural moments)

## How to Use This Document

- **Search:** Open `bitcoin_history_reference.md` and use Ctrl+F / Cmd+F to find any name, organization, or topic
- **Browse by tier:** Entries are organized roughly by tier and category within each tier
- **View change history:** Click **History** at the top of the file to see every update and what was added or changed

## Editorial Standard

This document is **factual and descriptive only** — no editorial opinion, superlatives, or invented connections. Entries describe what people did, not whether it was good or important. See [CONTRIBUTING.md](CONTRIBUTING.md) for the full standard.
